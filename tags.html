Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header make a header for important text
div division tag creates division or section
span span group inline elements in document
ol ordered list makes an ordered list
ul unordered list makes an unordered list
li list item makes an list item
blockquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong introduces imporant text <strong>technical sophistication</strong> technical sophistication
img image display image in html page <img></img>
code code format as source code <code>table</code> table
a anchor make hyperlink <code> Learn Enough</code> Learn Enough
span span define inline section in document <code>Call me Ishmael.</code> Call me Ishmael.

Document Tags

Tag Name Purpose Example Result
html html hypertext markup language root element <html>html</html> html
head head where metadata is introduced to page <head>head</head> head
body body main content area <body>body</body> body
title title display title of webpage <title>title</title> title
meta meta information about html <meta>meta</meta> meta
